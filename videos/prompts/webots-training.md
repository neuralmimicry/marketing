# Webots Training Video Prompt

Create an operator-training draft for Webots.

Output sections in this order:
1. Training objective
2. Audience and prerequisites
3. 4-5 minute walkthrough script
4. Step-by-step on-screen sequence
5. Operator checks after each major step
6. Terminology glossary for captions and transcript review
7. Risks or unknowns to verify before recording

Ground the walkthrough in this sequence:
- confirm login and token readiness
- choose the world profile and workspace label
- submit the launch handoff
- verify the browser runtime after redirect
- explain how the user should connect the session back to AARNN work

Avoid:
- skipping auth prerequisites
- vague demo narration with no practical checks
