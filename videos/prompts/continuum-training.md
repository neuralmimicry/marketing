# Continuum Training Video Prompt

Create an operator-training draft for Continuum.

Output sections in this order:
1. Training objective
2. Audience and prerequisites
3. 6-8 minute walkthrough script
4. Step-by-step on-screen sequence
5. Operator checks after each major step
6. Terminology glossary for captions and transcript review
7. Risks or unknowns to verify before recording

Ground the walkthrough in this sequence:
- set up or explain connection context
- review the adaptive loop and recent actions
- show a recruitment or placement decision
- explain what evidence the operator uses before and after action
- close with security and compliance posture, not just throughput

Avoid:
- dashboard-only narration
- assuming the user already understands auth, privilege, or endpoint context
