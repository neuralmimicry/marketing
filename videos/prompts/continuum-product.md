# Continuum Product Video Prompt

Create a commercial product-video draft for Continuum.

1. Video angle in one sentence
2. 90-120 second voiceover script
3. Shot-by-shot storyboard with on-screen text
4. Capture list tied to real control-plane or website surfaces
5. CTA slide copy
6. Facts to verify before recording

- Continuum is a sovereign hybrid control plane.
- Use plan-ramp-optimize-repeat as a core phrase.
- Show CLI plus guarded API, not just a dashboard.
- Mention Tracey-fed telemetry and operator-selectable placement policy.
- Include recruitment as a concrete differentiator.

- generic cloud-management copy
