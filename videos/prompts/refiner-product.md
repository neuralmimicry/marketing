# Refiner Product Video Prompt

Create a commercial product-video draft for Refiner.

Output sections in this order:
1. Video angle in one sentence
2. 90-120 second voiceover script
3. Shot-by-shot storyboard with on-screen text
4. Capture list tied to real product surfaces
5. CTA slide copy
6. Facts to verify before recording

Ground the draft in these themes:
- Refiner is a governed engineering workflow platform, not a generic assistant.
- The workflow router matters: analysis, research, project solving, and delivery are explicit choices.
- Use plan, act, verify, reflect as the clearest explanation of the agentic loop.
- Mention the shared service surface: jobs, auth, OIDC, RAG, MCP, voice, tokens, metrics, API docs.
- The close should point the viewer toward a Refiner briefing.

Avoid:
- language about replacing teams
- unsupported claims about autonomous deployment without review
- vague UI-tour narration with no business problem
