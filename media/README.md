# Media Workspace

This folder is for approved and in-progress media assets once recording and editing begin.

Recommended per-product substructure when assets start landing:
- `raw/`
- `edits/`
- `exports/`
- `captions/`
- `thumbnails/`

Suggested naming pattern:
- `YYYYMMDD-<product>-<video-type>-vNN.ext`

Current product folders:
- `media/refiner/`
- `media/continuum/`
- `media/tracey/`
- `media/aarnn/`
- `media/webots/`
